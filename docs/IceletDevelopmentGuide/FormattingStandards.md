# 小冰岛开发指北

## 1. 格式规范

### 角色命名

一般的，角色名使用**小写英文**来表达该角色的用途。当一个英文单词不足以说明该角色的用途时，使用**英文连字符**表示两个词语的连接，**不使用**下划线表示两个词语的连接。

### 造型命名

一般的，造型名使用**小写英文**来表达该造型的用途。当一个英文单词不足以说明该造型的用途时，使用**英文连字符**表示两个词语的连接，**不使用**下划线表示两个词语的连接。特别的，如果是出于为造型编号命名，则命名规则为使用**小写英文**+**英文连字符**+**数字**的格式。

### 声音命名

一般的，声音名使用**小写英文**来表达该声音的用途。当一个英文单词不足以说明该声音的用途时，使用**英文连字符**表示两个词语的连接，**不使用**下划线表示两个词语的连接。

### 变量与列表命名

一般的，变量与列表名使用**小写英文**来表达该变量与列表的用途。当一个英文单词不足以说明该变量与列表的用途时，使用**英文连字符**表示两个词语的连接，**不使用**下划线表示两个词语的连接。特别的，对于值仅为真和假的变量，将表示用途的词放在前面，将表示判断的词放在后面并加**英文问号**。命名规则为使用**表示用途的词**+**英文连字符**+**表示判断的词**+**英文问号**的格式。特别的，为便于快速辨认变量与列表，可在表示用途的词前添加**表情符号**。

### 消息命名

一般的，消息名使用**小写英文**来表达该消息的用途。当一个英文单词不足以说明该消息的用途时，使用**英文连字符**表示两个词语的连接，**不使用**下划线表示两个词语的连接。特别的，为便于快速辨认消息，可在表示用途的词前添加**表情符号**。

### 控制台输出信息命名

一般的，输出信息名使用**小写英文**和**大写英文**来表达该输出信息。当一个英文单词不足以表达该输出信息时，使用**英文空格**表示两个词语的连接，但对于连词仍使用**英文连字符**，**不使用**下划线表示两个词语的连接。
