# 小冰岛开发指北

## 2. 变量定义集

> [!TIP]
> 推荐使用“**查找**”功能在本页面查询你需要了解的变量。

`🧊os-name`
  - 适用于：所有角色
  - 值类型：首字母大写的英文单词
  - 是否为云变量（云变量存储位置）：是（本地）
  - 用途：存储玩家操作系统名称

`🧊os-ram`
  - 适用于：所有角色
  - 值类型：无符号整型
  - 是否为云变量（云变量存储位置）：是（本地）
  - 用途：存储玩家操作系统运行内存大小

`🧊output-console-show?`
  - 适用于：所有角色
  - 值类型：布尔型
  - 是否为云变量（云变量存储位置）：是（本地）
  - 用途：存储是否显示控制台输出信息

`🧊time`
  - 适用于：所有角色
  - 值类型：特殊型（两位无符号整形+冒号+两位无符号整形+冒号+两位无符号整形）
  - 是否为云变量（云变量存储位置）：是（本地）
  - 用途：存储玩家当前时区的时间

`output-quantity`
  - 适用于：所有角色
  - 值类型：无符号整数
  - 是否为云变量（云变量存储位置）：否
  - 用途：存储控制台输出信息的数量

`scrolling-y`
  - 适用于：所有角色
  - 值类型：无符号整数
  - 是否为云变量（云变量存储位置）：否
  - 用途：存储控制台可视化界面可被滑动的高度

`clone-is?`
  - 适用于：角色`output-console`
  - 值类型：布尔型
  - 是否为云变量（云变量存储位置）：否
  - 用途：判断正在打印控制台输出信息的是否为克隆体

`output-console-id`
  - 适用于：角色`output-console`
  - 值类型：无符号整数
  - 是否为云变量（云变量存储位置）：否
  - 用途：存储克隆体的编号数据

`text`
  - 适用于：角色`output-console`
  - 值类型：特殊型（字符串）
  - 是否为云变量（云变量存储位置）：否
  - 用途：存储当前控制台输出信息的所有内容
